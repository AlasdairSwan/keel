Backbone Architecture Demo

An example of a preferred Backbone.js architecture.

This example includes both compiled and uncompiled versions (before and after the grunt (v0.4) build is run:

  • Uncompiled Source
  • Compiled Distribution

Note: If your web root is this directory, the links above should work. If this backbone-arch-demo is a subdirectory of your web root, changing line 14 of /src/app/app.js to root: "/backbone-arch-demo/src/" should make the uncompiled source work (if this is multiple levels below the web root, all of those levels must be included). Running grunt (assuming it is installed) in the directory will build this change out to the compiled distribution, which should then work as well.

The purpose, however, is not to look at how beautiful this site is, but read the source code, which can be downloaded in its entirety at https://github.com/bobholt/backbone-arch-demo.
