place

  • [position='last']
)
chainable

Defined in src/framework/BaseView.js:294

Once the view has been rendered, it still needs to be placed in the document. The place method allows you to specify a destination for the view; this destination can either be a jQuery object, a DOM node, or a selector. The place method also optionally takes a position argument, which determines how the object will be placed in the destination node: as the first, last, or only child.

Parameters:

  • node HTMLElement

    The DOM element in which to place this view's element

  • [position='last'] String optional

    Identifier of where in the node to place the view's element

Example:

BaseView.place( $('body').get(0), 'last' );

elements

Array

Defined in src/framework/BaseView.js:64

If you would like to store references to certain elements in your template for later use, you can indicate those elements by doing both of the following:

  • adding a classname beginning with js- to the elements in your template
  • listing the classname suffix in your view's elements array

For example, if your template contains the following:

<div class="js-interesting"></div>

And your view's elements array is:

[ 'interesting' ]

Then your view will have a property interestingElement that references a jQuery object for the div.
